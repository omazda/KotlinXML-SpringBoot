Spring Boot XML Import

Панель импорта студентов из XML

Веб-приложение загружает XML-файл, сохраняет импорт в PostgreSQL и сразу показывает его в динамической таблице без перезагрузки страницы.

Spring Boot JAXB PostgreSQL Dynamic Table

Поддерживается тот же формат `students/student/skills`, что и в исходном консольном приложении.
